## Quickstart

Personal website running on Gatsby.Js, React.Js, and Node.js improved from [Gatsby Starter Blog](https://github.com/gatsbyjs/gatsby-starter-blog).

To install

```
yarn install
```

then run local server:

```bash
yarn
yarn run dev
```

Navigate to `localhost:8000`.
